Autonomous Robot

Required Robot Hardware

  • Drivetrain Assembly with two DC motors
  • Two wheels and one passive wheel
  • Small L298 Driver
  • Raspberry Pi
  • IR Sensors
  • Ultrasonic Sensors - triple sensor configuration
  • Magnetometer
  • Electromagnet
  • Electromagnet
  • One external rechargeable batter

This project involved implementing an autonomous navigation system using Python for a robot operating in a Manhattan-style grid environment. The system integrated real-time sensors(IR, ultrasound, and magnetometer), path planning, and real-time obstacle avoidance to safely explore the unknown grid-based map.

To handle real-time decision making, path planning, and error recovery, we integrated a multi-threaded software. The multi-threated software resulted in a code architecture including the following threads:

  1. Navigation Thread
  2. Sensor Reading and interpretation thread
  3. Graph/Map Thread (using matplotlib)
  4. ROS Communication Thread - for remote operation

This logic allowed for successful goal-directed navigation, robot exploration, replanning. As the robot navigated or explored, it constructed a map of its environment, planned efficient routes to goals, and switched seamlessly between manual and autonomous control.

Some of the key technical components include:

  • Line following [Navigation] which allowed for line detecting and intersection recognition.
  • Obstacle avoidance which prevented running into obstacles and signaled for replanning when necessary.
  • Dijkstra-Based exploration which allowed for optimal path detection

At the end, the robot demonstrated navigating accuracy, successful obstacle handling, efficient exploration, and robust system reliability.
