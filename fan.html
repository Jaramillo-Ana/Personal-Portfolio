Autonomous Fan

Project title: Smart Fan

This project involved building an automated system desinged to regulate a fan, based on room temperature. It uses a temperature sensor to monitor conditions and a relay module to control the fan, ensuring optimal comfort and energy efficiency. The inspiration of this project was energy efficient cooling systems and user-friendly devices and our goal was to build a user-friendly, low cost system that automates cooling based on environmental conditions.

The key components of this project were the temperature sensor (BMO688) which obtained temperature, humidity, gas, and pressure data, and a display which would display temperature to users.
